# ❤️ CardioCheck Pro AI

**CardioCheck Pro** is a machine learning-powered web application that predicts the risk of heart disease in patients based on 13 clinical features.

### 🚀 Features
* **AI Engine:** Powered by a Random Forest Classifier with **84% accuracy**.
* **Interactive Dashboard:** Built with Streamlit for real-time risk assessment.
* **Confidence Metrics:** Shows the probability percentage of the AI's prediction.
* **Clinical Reports:** Generates a downloadable PDF report for medical records.

### 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Scikit-Learn, XGBoost, Streamlit, Pandas, FPDF
* **Deployment:** Streamlit Community Cloud

### 📋 How to Use
1. Enter patient data in the left sidebar (Age, Cholesterol, BP, etc.).
2. Click **"Run Diagnostic Analysis"**.
3. View the risk level and confidence score.
4. Download the PDF report if needed.
